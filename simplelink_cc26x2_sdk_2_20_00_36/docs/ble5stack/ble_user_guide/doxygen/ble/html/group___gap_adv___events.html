GapAdv Events
Gap Advertiser

Events sent from the GapAdv module

See GapAdv Event IDs for the following events which are related to this module:

  • GAP_EVT_ADV_START_AFTER_ENABLE
  • GAP_EVT_ADV_END_AFTER_DISABLE
  • GAP_EVT_ADV_START
  • GAP_EVT_ADV_END
  • GAP_EVT_ADV_SET_TERMINATED
  • GAP_EVT_SCAN_REQ_RECEIVED
  • GAP_EVT_ADV_DATA_TRUNCATED
  • GAP_EVT_INSUFFICIENT_MEMORY

These are set with GapAdv_setParam and read with GapAdv_getParam
