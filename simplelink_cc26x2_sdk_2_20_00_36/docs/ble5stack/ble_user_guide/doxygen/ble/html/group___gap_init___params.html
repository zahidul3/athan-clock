§ GapInit_PhyParamId_t

enum GapInit_PhyParamId_t

GAP Initiator PHY Parameters

These can be set with GapInit_setPhyParam and read with GapInit_getPhyParam. The default values below refer to the values that are set at initialization.

Enumerator
INIT_PHYPARAM_SCAN_INTERVAL 

Scan Interval

default: INIT_PHYPARAM_DFLT_SCAN_INTERVAL

range: 4 - 16384

INIT_PHYPARAM_SCAN_WINDOW 

Scan Window

default: INIT_PHYPARAM_DFLT_SCAN_INTERVAL

range: 4 - 16384

INIT_PHYPARAM_CONN_INT_MIN 

Minimum Connection Interval

Note
This should be equal to or smaller than INIT_PHYPARAM_CONN_INT_MAX

default: INIT_PHYPARAM_DFLT_CONN_INT_MIN

range: 6 - 3200

INIT_PHYPARAM_CONN_INT_MAX 

Maximum Connection Interval

Note
This should be equal to or greater than INIT_PHYPARAM_CONN_INT_MIN

default: INIT_PHYPARAM_DFLT_CONN_INT_MAX

range: 6 - 3200

INIT_PHYPARAM_CONN_LATENCY 

Slave Latency

default: INIT_PHYPARAM_DFLT_CONN_LATENCY

range: 0 - 499

INIT_PHYPARAM_SUP_TIMEOUT 

Supervision Timeout

default: INIT_PHYPARAM_DFLT_SUP_TIMEOUT

range: 10 - 3200

INIT_PHYPARAM_MIN_CE_LEN 

Minimum Length of Connection Event

Warning
This is not used by the controller

default: INIT_PHYPARAM_DFLT_MIN_CE_LEN

range: 0 - 0xFFFF

INIT_PHYPARAM_MAX_CE_LEN 

Maximum Length of Connection Event

Note
This should be equal to or greater than INIT_PHYPARAM_DFLT_MIN_CE_LEN
Warning
This is not used by the controller

default: INIT_PHYPARAM_DFLT_MAX_CE_LEN

range: 0 - 0xFFFF
