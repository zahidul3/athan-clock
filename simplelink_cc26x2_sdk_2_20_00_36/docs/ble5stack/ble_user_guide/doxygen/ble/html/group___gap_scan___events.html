GapScan Events
Gap Scanner

Events sent from the GapScan module

See GapAdv Event IDs for the following events which are related to this module:

  • GAP_EVT_SCAN_ENABLED
  • GAP_EVT_SCAN_DISABLED
  • GAP_EVT_SCAN_PRD_ENDED
  • GAP_EVT_SCAN_DUR_ENDED
  • GAP_EVT_SCAN_INT_ENDED
  • GAP_EVT_SCAN_WND_ENDED
  • GAP_EVT_ADV_REPORT
  • GAP_EVT_ADV_REPORT_FULL
  • GAP_EVT_INSUFFICIENT_MEMORY

These are set with GapScan_setParam and read with GapScan_getParam
