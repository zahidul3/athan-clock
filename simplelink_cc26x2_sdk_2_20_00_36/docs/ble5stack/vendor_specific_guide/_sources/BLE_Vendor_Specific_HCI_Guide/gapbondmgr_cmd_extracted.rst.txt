:Command Description: Set the privacy mode for a remote device

**Command Parameters**

:addrType: Configure the device's identity address type.

 :Size: 1 byte(s)
 :Default: 0x00
 :Range:
         ===================== ========================================================

         ===================== ========================================================

 :Size: 6 byte(s)

 :Size: 1 byte(s)

         ===================== ========================================================
